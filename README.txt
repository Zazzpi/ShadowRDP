ShadowRDP+ — Менеджер RDP-сессий

Описание:
Приложение предназначено для просмотра и управления активными RDP-сессиями на серверах.

Функционал:

* Просмотр активных RDP-сессий с нескольких серверов
* Подключение к сессии пользователя (shadow)
* Завершение сеанса (logoff)
* Принудительный сброс сеанса (reset)
* Поиск по пользователю или серверу
* Контекстное меню (ПКМ)
* Горячие клавиши

---

Запуск:

1. Открыть папку ShadowRDP
2. Запустить файл ShadowRDP.exe

---

Настройка серверов:
Файл servers.txt находится в папке:
ps_scripts\servers.txt

Пример содержимого:
172.19.10.10
172.19.10.11
server-name

Каждый сервер указывается с новой строки.

---

Горячие клавиши:
Enter           — подключиться к пользователю
Delete          — завершить сеанс
Shift + Delete  — принудительный сброс сеанса
F5              — обновить список
Ctrl + F        — перейти в поиск
Esc             — очистить поиск

---

Контекстное меню (ПКМ):

* Подключиться
* Завершить сеанс
* Принудительная перезагрузка сеанса
* Обновить список

---

Описание действий:

Подключиться:
Подключение к сессии пользователя (режим shadow)

Завершить сеанс:
Корректное завершение сессии пользователя (аналог выхода из системы)

Принудительная перезагрузка:
Жёсткое завершение сессии

---

Требования:


Права администратора (или соответствующие права на RDP)
